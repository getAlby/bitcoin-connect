# dev/vite

Develop Bitcoin Connect components using [Vite](https://vitejs.dev/)

## Install

Run `yarn install` here and in the root directory.

## Development

Run `yarn dev` (either here or in the root directory)


> Note: To make testing easy, these examples use a test Lightning address. Invoices can't be paid with real sats; please use sats from the faucet: https://faucet.nwc.dev
