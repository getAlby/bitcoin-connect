# dev/vite

Develop Bitcoin Connect components using [Vite](https://vitejs.dev/)

## Install

Run `yarn install` here and in the root directory.

## Development

Run `yarn dev` (either here or in the root directory)


> Note: To make testing easy, the examples here use a faucet rather than real Bitcoin. You can grab some free testing sats from https://faucet.nwc.dev to get started.
